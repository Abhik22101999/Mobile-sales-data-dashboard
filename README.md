# Mobile-sales-data-dashboard

📊 Mobile Sales Data Dashboard – Power BI

This repository contains my Mobile Sales Data Dashboard created in Microsoft Power BI, showcasing key insights from mobile sales transactions. The project was built as part of my learning journey under the guidance of Satish Dhawale Sir, Founder of Skill Course.

🚀 Project Overview

The Mobile Sales Data Dashboard provides an interactive and visual analysis of mobile sales performance, helping stakeholders track trends, monitor KPIs, and make data-driven decisions.

Key Features

Total Sales, Quantity & Transactions Overview

Monthly & Daily Sales Trends with drill-down capabilities

Sales by Mobile Model (e.g., Galaxy Note 20, Vivo Y51, OnePlus Nord)

Geographical Sales Distribution by City

Payment Method Analysis (Cash, UPI, Credit Card, Debit Card)

Day-wise Sales Performance

🛠 Tools & Technologies

Microsoft Power BI – Dashboard creation & data visualization

Data Source – Mobile sales dataset (sample data)

Data Modeling – Power Query, DAX measures

📚 Learning Outcomes

Download the .pbix file from this repository

Open it in Microsoft Power BI Desktop

Explore the visuals and interact with slicers/filters
